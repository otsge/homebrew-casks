# OTSGE Casks

## How do I install these formulae?

`brew install otsge/casks/<formula>`

Or `brew tap otsge/casks` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "otsge/casks"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
